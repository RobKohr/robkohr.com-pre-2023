SEO friendly ajax links

ConstantSail is going to be available in multiple languages, and I put an image button in the menu which will pop up a list of links to the urls of the translated sites (example: http://contantsail.com/lang-es/ for spanish).

The popup is a javascript reveal of a div (ok so it isn’t really ajax), but I wanted google’s crawler to actually go to the page with the list of links to the translations.

What I did was create a link with the href pointing to the url I wanted google to go, and an onclick event that did the popup and return false (so that it doesn’t follow the href). If this would be pulled from actual ajax, this would be even more helpful since google can index all that fine content that would have been hidden behind the ajax call.

Code:

<a href=“/languages/” onclick=“$(’#languages’).show();return false;”>
